### Pré-requisitos
#
Para o desenvolvimento certifique-se de ter o [NodeJS] no seu sistema.
### Instalação
Instalar NodeJS
https://nodejs.org/en/
Checar Versão
```sh
node -v
```

Instalar NPM
```sh
npm i
```

Checar Versão
```sh
npm -v
```

### Webpack
#
```sh
npm run dev
```

```sh
npm run prod
```
